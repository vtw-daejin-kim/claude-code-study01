## 🎯 핵심 기능 요구사항

### 1. 사용자 관리
- **회원가입 / 로그인**
  - JWT 기반 인증 시스템
  - 이메일/비밀번호 기반 인증
  - 소셜 로그인 (선택사항)
  
- **사용자 프로필 관리**
  - 개인정보 수정
  - 비밀번호 변경
  - 프로필 이미지 업로드
  
- **주소록 관리**
  - 배송지 주소 등록/수정/삭제
  - 기본 배송지 설정
  
- **권한 관리**
  - 일반 사용자 (Customer)
  - 관리자 (Admin)

### 2. 상품 관리
- **상품 카테고리**
  - 카테고리 분류 체계
  - 카테고리별 상품 조회
  
- **상품 목록 조회**
  - 검색 기능 (상품명, 설명)
  - 필터링 (가격, 카테고리, 브랜드 등)
  - 정렬 (인기순, 최신순, 가격순)
  - 페이징 처리
  
- **상품 상세 정보**
  - 상품 이미지 (다중 이미지 지원)
  - 상품명, 가격, 설명
  - 재고 수량
  - 옵션 (색상, 사이즈 등)
  
- **상품 등록/수정/삭제 (관리자)**
  - 상품 정보 입력
  - 이미지 업로드
  - 재고 관리

### 3. 쇼핑 기능
- **장바구니**
  - 상품 추가/삭제
  - 수량 변경
  - 선택 상품 주문
  
- **위시리스트 (찜하기)**
  - 관심 상품 저장
  - 장바구니로 이동
  
- **상품 리뷰/평점**
  - 구매 확정 후 리뷰 작성
  - 별점 평가 (1-5점)
  - 리뷰 이미지 첨부
  - 리뷰 수정/삭제

### 4. 주문/결제
- **주문 생성**
  - 배송지 선택
  - 주문 상품 확인
  - 총 금액 계산
  
- **결제 시스템**
  - 결제 방법 선택
  - 결제 처리
  - 결제 완료 알림
  
- **주문 내역 조회**
  - 주문 목록 확인
  - 주문 상세 정보
  - 주문 상태 추적
  
- **주문 상태 관리**
  - 주문 접수
  - 결제 완료
  - 배송 준비 중
  - 배송 중
  - 배송 완료
  - 구매 확정
  - 취소/환불

### 5. 재고 관리
- **재고 추적**
  - 실시간 재고 수량 업데이트
  - 주문 시 재고 차감
  - 취소 시 재고 복구
  
- **재고 알림**
  - 재고 부족 경고
  - 품절 상태 표시

### 6. 관리자 기능
- **관리자 대시보드**
  - 주요 지표 표시 (매출, 주문 수, 회원 수 등)
  - 최근 주문 내역
  - 통계 차트
  
- **주문 관리**
  - 전체 주문 목록 조회
  - 주문 상태 변경
  - 주문 취소/환불 처리
  
- **사용자 관리**
  - 회원 목록 조회
  - 회원 정보 조회/수정
  - 회원 활동 정지
  
- **통계/분석**
  - 매출 통계
  - 인기 상품 분석
  - 카테고리별 판매 현황
